# Health Ops Roadmap

Implemented foundation:

1. Detailed validation and coherence issue payloads in dashboard health.
2. Dashboard rendering of issue messages and suggestions.
3. Action buttons on issues that navigate users to the most relevant workspace.
4. Persisted health snapshots with a short history trend.

Next tranche:

1. Convert issue cards into one-click mutations:
   - create action item
   - create task
   - create risk
   - assign reviewer
2. Add issue lifecycle:
   - dismiss
   - snooze until date
   - accept with rationale
3. Add scoped checks:
   - by WP
   - by deliverable
   - by reporting period
4. Add scheduled and event-driven reruns.
5. Add trend analytics and recurring-issue detection.
